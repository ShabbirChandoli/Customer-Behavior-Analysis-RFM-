# Customer-Behavior-Analysis-RFM-

This Power BI project analyzes customer purchasing behavior using the RFM (Recency, Frequency, Monetary) model.

The dataset was cleaned in Power Query by removing cancelled orders, invalid transactions, and missing Customer IDs before creating the RFM model.

The dashboard segments customers into Champions, Loyal Customers, Potential Loyalists, At Risk, Lost Customers, and Others.

It includes KPIs, revenue by customer segment, customer distribution, and a customer behavior scatter plot to identify high-value and inactive customers.

Tools Used: Power BI, Power Query, DAX, Excel.
